# Audience Messaging: The Message Matrix

**Executive summary.** One policy, seven audiences, seven distinct messages — each pairing a core promise with the right messenger, a concrete proof point, and the specific landmine that loses that audience permanently. This file operationalises the coalition map in [06-palatability/README.md](../06-palatability/README.md); messenger recruitment itself belongs to [13-movement/messengers.md](../13-movement/messengers.md).

## Principles

1. **Segmented messages, single doctrine.** Every message below leads with the endowment and uses the fixed number; segmentation varies the *frame around* the doctrine, never the doctrine ([README.md](README.md)).
2. **Messenger beats message.** Polling on trust consistently shows in-group messengers outperform experts for contested claims (to verify against current UK trust surveys). A farmer defending the farm carve-out is worth ten economists.
3. **Landmines are permanent.** A message that hits a segment's landmine doesn't merely fail — it supplies the opposition a clip ([10-opposition/attack-playbook.md](../10-opposition/attack-playbook.md)).

## Cross-segment collision management

Segmented messaging risks being quoted across segments ("they tell pensioners one thing and renters another"). Defences:

- **All messages are true simultaneously.** Segmentation selects emphasis, never contradicts. The threshold genuinely protects the homeowner *and* the endowment genuinely funds the renter; there is no private register.
- **Assume every message is public.** Draft each cell above as if it will be read aloud to the most hostile other segment. The matrix above passes that test by construction; future additions must too.
- **The doctrine line is the tiebreak.** When segments conflict on emphasis, the endowment-first rule wins ([README.md](README.md)).
